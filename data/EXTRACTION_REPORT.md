## Data Fields Extracted

Each district includes:

1. **Basic Info**
   - `code` - District code (e.g., R-1, C-2)
   - `name` - Full district name
   - `category` - Classification (Residential, Commercial, Industrial, etc.)

2. **Dimensional Standards**
   - `min_lot_sqft` - Minimum lot size in square feet
   - `min_lot_width_ft` - Minimum lot width
   - `max_height_ft` - Maximum building height
   - `max_stories` - Maximum number of stories
   - `coverage_pct` - Maximum lot coverage percentage

3. **Setbacks**
   - `front_setback_ft` - Front yard setback
   - `side_setback_ft` - Side yard setback
   - `rear_setback_ft` - Rear yard setback

4. **Density**
   - `density` - Units per acre (residential only)

5. **Metadata**
   - `source_url` - Original source link
   - `verified_date` - Date of verification
