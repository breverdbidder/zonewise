# ZoneWise Strategic Positioning

> **Positioning:** Agentic AI ecosystem (NOT SaaS) for zoning intelligence

## The Six Human Needs Framework

| Need | User Want | ZoneWise Delivery |
|------|-----------|-------------------|
| **CERTAINTY** | Will my project get approved? | Real ordinance data, source citations, confidence scores |
| **VARIETY** | Evaluate alternatives faster | Multi-jurisdiction search, 10 parcels in seconds |
| **SIGNIFICANCE** | Be the zoning expert | AI-augmented expertise, "ZoneWise Pro" badge |
| **CONNECTION** | Trusted data source | Brevard insider credibility, community corrections |
| **GROWTH** | Expand practice, complex projects | Historical variance database, pattern recognition |
| **CONTRIBUTION** | Build community-improving projects | Enable compliant development, reduce variance battles |

**Marketing insight:** Lead with CERTAINTY (fear) + SIGNIFICANCE (ego).

## Identity Preservation Messaging

**User fear:** AI will replace me as a zoning consultant  
**ZoneWise promise:** We make YOU the AI-augmented expert, not the obsolete one

> "ZoneWise doesn't replace your expertise—it amplifies it. You're not competing with AI. You're the professional who wields it."

## Competitive Moats

1. **Data Moat:** 189 districts with REAL ordinance data (competitors use synthetic)
2. **Distribution Moat:** MCP-native, inside Claude/Cursor—no app download friction
3. **Network Moat:** User corrections improve data; more users = better accuracy
