## Enhanced Features Documented

Each zoning district includes the following enhanced dimensional standards:

### Core Dimensional Requirements
- **Min Lot Size** (sq ft)
- **Min Lot Width** (ft)
- **Max Height** (ft)
- **Max Coverage** (%)

### Setback Requirements
- **Front Setback** (ft)
- **Rear Setback** (ft)  
- **Side Setback** (ft)
- **Corner Setback** (ft) - for corner lots

### Additional Standards
- **Max Stories** - building height in stories
- **Parking Residential** - spaces per dwelling unit
- **Parking Commercial** - spaces per 1,000 sq ft
- **Floor Area Ratio (FAR)** - for commercial zones
- **Density Max** - dwelling units per acre
- **Min Dwelling Size** (sq ft)

## API Access

**Base URL:** https://zonewise-api-v3.onrender.com

**Endpoints:**
- `GET /health` - Service health check
- `GET /api/v1/jurisdictions` - List all jurisdictions
- `POST /api/v1/analyze` - Property compliance analysis
- `POST /api/v1/compliance` - Full CrewAI multi-agent analysis

**Example Request:**
```bash
curl -X POST https://zonewise-api-v3.onrender.com/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{"address": "798 Ocean Dr, Satellite Beach, FL"}'
```
